1) git init (to initialize the git empty repository)
2) git status (to check the status)
3) Create a sample file and add
   
	git add index.html (To add single file)
		OR
	git add . (To add multiple files)
4) git commit -m "Message" (to commit the file with a message)
5) git status
6) git log (to check the logs like how many commits,etc.)
7) git log --oneline (to check the logs in one line)
8) git config --global user.name "apurvaseth"
   git config --global user.email "apurva.seth@capgemini.com"
9) git branch newFeature (to create a new branch)
   git branch (to list the branches)
10) git checkout newFeature (switches to different branch)
